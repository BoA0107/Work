1. Account Code Setup

1.1. My Workspace –> Account Manage –> System Setup –> Account Management Rules.
1.2. Account Codes标签页 –> Create Account Code.
1.3. 输入对应内容：
Account_Code_Setup
1-Account Code Account Code 随意4个数字
Shot Description 随意
Description 随意
2-Apply Batch Type Cash Payment
3-Apply Balance Group Service Pay
4-Main General Ledger Company Company
Business Unit 0001-Payable
Object 400112-other commission
Subsidiary 037-037
5-Offset General Ledger Company Company
Business Unit 0001-Payable
Object 110101-social insurance
Subsidiary 004-004

Sequence Bottom

Value 100
Value Description ABCDEFG


3. Import Batches
3.1. My Workspace –> Account Manage –> Import Batch.
3.2. 输入对应内容：
Import Batches
Import Category Account Transaction
Import Type GenericImport
Import File GenericImport.xlx 文件（文件名与以前上传的不能重复）
Comments 随意
Requested By 随意

3.3 点击 Import后，记录下BatchID
4. Create Batch
4.1. My Workspace –> Account Manage –> View Batch Transactions.
4.2. Create Batch –> Batch Type, Comments, Requested By 随便填。
(点击 Import后，记录下BatchID)


5. Write Off Batches
5.1. My Workspace –> Account Manage –> Write off Transactions Management
5.2. Balance Range 填写 From 0.01 to 0.02，点击 “Search” 查找并记录下Consultant Number (可能搜不到，此时将From 至 to 的范围放大，直到能找出为止)。

5.3. 点击“Next” –> Batch Type 填写 “Write Off General Revenue”；Transaction Description 和 Comments 随便填写 –> 点击 “Finish” –> 生成并记录下 BatchID。

5.4. 4. My workspace输入搜索 “View Batch Transactions” –> “Unposted Batch” tab中找到上一步的batch 并审批通过，之后该batch 进入 “Ready for post Batch” tab –> 用另一个帐号的 My workspace输入搜索 “View Batch Transactions” –> Ready for Post Batch,双击该batch 去 “Post batch”.


6. Write Off Reinstate
6.1. 找到5步搜索出的consultant 人员的工作台 –> 搜索 Account Management –> Account Write Off Reinstate –> 点击 “Manage”下面文件图标 –> 点击 “Create Case”
6.2. Account Code ，填写本次新建的Account Code，然后提交审核，不用确认”pending Approval”状态就好。
6.3. 去 consultant下 –> Account Transaction 查看并记录 Transaction ID。





7. Delete Batch
7.1. My Workspace –> Account Manage –> View Batch Transactions ，在Unposted batch 里找一个本次新建的batch ，点击 Delete Batch –> 该batch 出现在 Deleted Batch里即可
7.2. Account Code ，填写本次新建的Account Code，然后提交审核，不用确认”pending Approval”状态就好。
7.3. 去 consultant下 –> Account Transaction 查看并记录 Transaction ID。
