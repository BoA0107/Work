BCA测试账号设置

1. 查找BC

SAM --> 高级搜索 --> 选择定 Activity Status 和 Career Level进行搜索
推荐：1. Activity Status为A状态；2. Career Level 75-85


2. 核对执照

1. BC workspace --> Business Director License Info 可以看到改BC名下所有执照 --> 点击查看执照具体信息
2. 打开国家企业信用信息公示系统 http://www.gsxt.gov.cn/corp-query-homepage.html查看该执行状态。



3. 终止合同

1. BC workspace --> Business Director License terminated by others
2. 选择要终止的执照，"执照有效截至日期"设置为当天以前的日期，"选择备注""备注（其他）"随意选择、填写，提交案例。
3. 该案例需要二次审批。
4. 在My workspace --> request job: "Job Type" == CCM ; Job Category == CCM BDL Expiration Work; Select Job(s) == BDLExpirationBuildWorkJob。 Job Run Time设置为"执照有效截至日期" --> 点击"Submit".
5. 可在"Job Status"界面查看job执行情况（Begin Date需早于Job Run Time时间）。
6. Job执行完后，可返回BC workspace的"Contact History"界面查看（会生成"Agreement terminated by BDL expired"案例并且状态为Completed）



4. 调整渠道 *

BC workspace --> Open License Request Change --> 选择角色提交案例即可。
如果需要的角色是"小微企业自有"或"直销员"，操作到此即可；如果需要的角色是"小微企业代办"需要后续操作



5. 收稻签约 *

----ebizfoawscnuat/consultants
select top 10 * from cn.CCM_VendorLicenseNameList
select top 10 * from cn.CCM_ShouDaoVendorLicenseResult
select top 10 * from cn.CCM_BPWorkShouDaoVendorLicenseResultSync

5.1. 初始状态

朱凤（contact ID：20003900805）为例，执照已终止，已选择"小微企业代办"
直销员：1；收稻代办：2；收稻自有：3
CCM_VendorLicenseNameList 有数据且"VendorLicenseType=2"
cn.CCM_ShouDaoVendorLicenseResult 无数据
cn.CCM_BPWorkShouDaoRegUsers 无数据


5.2. 注册收稻

如下图设置并执行Request Job，执行后再次查询数据表

CCM_VendorLicenseNameList 有数据且"VendorLicenseType=2"
cn.CCM_ShouDaoVendorLicenseResult 有数据且status=1
cn.CCM_BPWorkShouDaoRegUsers 有数据且StatusID=3


5.3. 签约成功

执行SQL,将cn.CCM_ShouDaoVendorLicenseResult表中status更新为2

update cn.CCM_ShouDaoVendorLicenseResult set StatusID=2 where contactid = 20003900805
最终各表结果应如下：
CCM_VendorLicenseNameList 有数据且"VendorLicenseType=2"
cn.CCM_ShouDaoVendorLicenseResult 有数据且status=2
cn.CCM_BPWorkShouDaoRegUsers 有数据且StatusID=3




6. BDL调整 *

1.根据BC执照编号在cn.BDA_BusinessDirectorLicense表中找到其BDLID
2.在cn.BDA_BusinessDirectorLicenseExtn表中把VendorLicenses设置为1
----查找对应BDLID
select * from cn.BDA_BusinessDirectorLicense where GovLicenceNumber='91110101MA003NQE6L'
----更新BDLID
update cn.BDA_BusinessDirectorLicenseExtn set VendorLicense=1 where BDLID='00AE27F6-3D61-4E8D-A918-3EC032CC06EF'
